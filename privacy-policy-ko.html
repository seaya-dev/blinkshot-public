깜빡샷 개인정보 처리방침

시행일: 2026년 5월 25일

깜빡샷은 사용자가 직접 선택하거나 촬영한 스크린샷에서 텍스트를 추출하고, 알림 카드를 만들 수 있도록 돕는 iPhone 앱입니다. 회원가입, 서버 업로드, 외부 분석 SDK를 사용하지 않습니다.

1. 수집 및 저장하는 정보

깜빡샷은 별도의 회원가입을 요구하지 않습니다. 사용자가 앱 안에서 저장한 이미지 사본, 추출된 텍스트, 카드 제목, 메모, 유형, 마감일, 알림 시간은 사용자의 기기 안에 저장됩니다.

2. 사진 접근

깜빡샷은 사용자가 직접 선택한 이미지에 대해서만 접근합니다. 사진첩 전체를 자동으로 스캔하거나 원본 사진을 수정 또는 삭제하지 않습니다.

3. 카메라와 캘린더

사용자가 카메라 촬영을 선택한 경우, 촬영한 이미지는 알림 카드 생성을 위해 앱 내부에 저장됩니다. 사용자가 캘린더에 추가를 선택한 경우에는 iOS의 캘린더 편집 화면에서 카드 제목, 날짜, 메모를 확인하고 직접 저장할 수 있습니다. 깜빡샷은 기존 캘린더 일정을 자동으로 읽거나 수정하지 않습니다.

4. OCR 처리

이미지 속 텍스트 추출은 Apple Vision 프레임워크를 사용해 기기 안에서 처리합니다. OCR 처리를 위해 이미지나 텍스트를 외부 서버로 전송하지 않습니다.

5. 알림

깜빡샷은 사용자가 직접 설정한 카드에 대해서만 로컬 알림을 예약합니다. 알림 권한은 사용자가 알림 시간을 설정할 때 요청됩니다.

6. 데이터 삭제

사용자는 앱의 설정 화면에서 저장된 카드와 앱 내부에 복사된 이미지 데이터를 삭제할 수 있습니다. 앱을 삭제하면 기기 안에 저장된 앱 데이터도 함께 삭제됩니다.

7. 제3자 제공

깜빡샷은 사용자의 이미지, OCR 텍스트, 카드 데이터를 제3자에게 제공하지 않습니다.
